ONTHEGO
Проект ONTHEGO — это универсальная геймификационная образовательная платформа для обучения навыкам кибербезопасности. Основная цель платформы – сделать процесс изучения кибербезопасности более увлекательным и эффективным за счёт игровых элементов (баллы опыта, значки, рейтинг пользователей и т.д.) и интерактивных сервисов (онлайн-курсы, тесты, флеш-карточки, режим «Помодоро», встроенный чат-ассистент и др.). ONTHEGO позиционируется как «Elite Security Training Platform» для подготовки квалифицированных специалистов по информационной безопасности.
Превью/демонстрация
•	Главная страница: содержит заголовок с логотипом, навигацию по разделам и интерактивный 3D-фон (анимация загружается через библиотеку Spline). Также есть кнопки для начала обучения и быстрого перехода к разделам «Программа курса» и «FAQ».
•	Раздел «О платформе»: краткое описание ключевых возможностей ONTHEGO и его преимуществ (использование геймификации, интеграция ИИ-ассистента и т.д.).
•	Страница «Контакты»: содержит контактную информацию и форму обратной связи.
•	Страница «Политика конфиденциальности»: стандартный текст политики (код HTML/CSS/JS находится в папке static/privacy).
•	Cookie Consent: отдельная страница cookie-system.html с интерфейсом для принятия политики использования cookie.
•	Скриншоты: изображения интерфейса (главной страницы, «О платформе», «Контакты» и др.) можно найти в папке ONTHEGO-photo (в архиве проекта).
Используемые технологии
•	HTML5 & CSS3: верстка сайта и стилизация (используются современные CSS-переменные, Flexbox и Grid).
•	JavaScript (ES6+): реализует интерактивность сайта (открытие/закрытие мобильного меню, обработка согласия cookie и др.).
•	Spline 3D Viewer: подключается библиотека @splinetool/viewer через CDN для отображения 3D-анимации на главной странице.
•	Google Fonts: подключены шрифты Inter и Space Grotesk для оформления текста.
•	Backend: отсутствует — проект представляет собой полностью статический сайт. В документации упоминалась планируемая многоуровневая архитектура с серверной частью, но в коде ее нет.
Как запустить проект локально
1.	Скачайте архив проекта и распакуйте его (или выполните git clone <URL репозитория>).
2.	В терминале перейдите в корневую папку проекта, затем в каталог static:
 	cd ONTHEGO-website/static
3.	Запустите локальный веб-сервер. Например, с помощью Python:
 	python3 -m http.server 8000
4.	Откройте в браузере адрес http://localhost:8000/index/index.html. Это загрузит главную страницу сайта ONTHEGO. Вы также можете открыть файлы напрямую, но рекомендуется использовать веб-сервер для корректной работы относительных ссылок и анимации.
Структура проекта
•	static/ – статические файлы фронтенда: HTML, CSS и JS. Внутри находятся подпапки для отдельных страниц (например, index/, about/, contacts/, privacy/) и файл cookie-system.html.
•	documents/ – техническая документация проекта: анализ требований, системная архитектура, технические спецификации, UI-прототип (ссылка на Figma).
•	ONTHEGO-photo/ – скриншоты и макеты интерфейса (скриншоты страниц сайта и прототипов).
•	Прочие файлы в корне: ONTHEGO-Presentation.pptx (презентация проекта), файлы изображений Системная_Архитектура_*.png/.jpg (диаграммы архитектуры), а также этот файл README.md.
•	.git/ – скрытая папка системы контроля версий Git (игнорируется при работе с проектом).
